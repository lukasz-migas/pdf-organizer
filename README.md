# PDF Organizer

An app for splitting, previewing, reordering, and merging PDF labels.

## Document type defaults

- Filenames starting with `Vinted` default to `Whole-page label`.
- Hash-like filenames made of mixed letters and digits default to `Bottom 40% rotated`.
- Everything else defaults to `A4 quadrants`.

These are only defaults. Each uploaded PDF can still be reassigned manually in the UI.
